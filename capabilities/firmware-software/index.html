Firmware Engineering

Custom Firmware Development

Our firmware engineers are core contributors to Marlin and active Klipper developers. We do not just configure existing firmware — we extend and optimize it for your specific hardware, motion system, and feature set.

Custom-compiled Marlin 2.x firmware for your hardware variant. Every pin, sensor, endstop, thermistor, stepper driver, and heater is mapped and validated. Klipper Deep Dive

Why Klipper for Professional Printers

Klipper's offloaded architecture (Raspberry Pi handles computation, MCU handles real-time motor control) enables features that are impractical on traditional 8-bit or 32-bit standalone firmware. Our team has deep Klipper expertise for partners targeting the professional and prosumer segments.

Klipper Advantages

  • ✓Input shaping: eliminates ringing/ghosting at high speeds through accelerometer-based resonance compensation
  • ✓Pressure advance: compensates for extruder pressure dynamics, producing crisp corners at any speed
  • ✓Smooth pressure advance: eliminates extruder gear artifacts on flexible filaments
  • ✓Web-based interface (Mainsail/Fluidd): browser-accessible from any device on the network
  • ✓Python-based macros: infinitely customizable automation without recompiling firmware
  • ✓Remote firmware updates: change printer.cfg without reflashing the MCU

Our Klipper Customizations

  • ✓Branded Mainsail/Fluidd theme with your logo and color palette
  • ✓Pre-calibrated input shaping and pressure advance values for your hardware
  • ✓Automatic ADXL345-based resonance measurement with one-click calibration
  • ✓Custom KlipperScreen theme for dedicated touchscreen displays
  • ✓Print farm manager: control multiple printers from a single dashboard
  • ✓AI failure detection integration (Obico/Spaghetti Detective compatible)

Slicer Profiles

Pre-Tuned Profiles for Every Slicer

Your customers should get great prints out of the box — not after weeks of trial and error. We develop and maintain optimized slicer profiles for all major slicing software platforms.

Cura Profiles

Complete Ultimaker Cura profile package with branded printer definition, custom start/end G-code, material-specific profiles for PLA, PETG, ABS, TPU, and specialty filaments. Quality presets (Draft, Standard, Fine, Ultra Fine) optimized for your hardware. Available as a .curaprofile file for one-click installation.

PrusaSlicer Profiles

PrusaSlicer configuration bundle with print, filament, and printer settings. Tested across PrusaSlicer stable and alpha releases. Includes organic/tree support settings calibrated for your hotend and cooling system. Multi-material painting presets for single-extruder filament changes.

Orca Slicer Profiles

Orca Slicer profiles with built-in calibration tests (temperature tower, pressure advance, flow rate, retraction, tolerance). Our profiles include pre-populated calibration values so users can skip the tuning process entirely. Optimized for Orca's unique features: scarf joint seam, adaptive pressure advance.

Profile Testing Methodology

Every slicer profile undergoes a standardized 12-test validation suite: Benchy (overall quality), temp tower (adhesion vs. stringing), retraction test, overhang test (0-80 degrees), bridging test, dimensional accuracy cube, tolerance gauge, all-in-one micro test, vase mode, and 24-hour endurance print. Test results documented per profile revision.

Profile Distribution & Updates

Profiles shipped on the printer's SD card, available for download from your branded support page, and submitted to slicer vendors for inclusion in official releases. We maintain profile versions and publish update changelogs when new materials or features are validated. Automatic profile update notifications via the companion app.

Developer Tools

API & SDK for Enterprise Partners

For partners building custom software on top of our hardware — education platforms, print farm operators, enterprise fleet managers — we provide comprehensive APIs and SDKs.

REST API

Full RESTful API with JSON payloads for printer management. Endpoints: GET /printer/status (temperature, position, progress), POST /printer/command (start, pause, stop, home, set temperature), POST /printer/file (upload G-code with metadata), GET /printer/camera (stream URL or snapshot), GET /printer/history (job log with statistics). Swagger/OpenAPI 3.0 documentation included. Bearer token authentication with configurable permissions.

SDK & Plugin System

Python SDK (pip-installable package) and JavaScript/TypeScript SDK (npm package) for building custom integrations. Includes connection manager, command builder, event listener, and file upload utilities. Plugin architecture for extending printer functionality: example plugins include cloud dashboard connector, print queue manager, filament inventory tracker, and automated QC image capture. Full developer documentation with code examples and tutorials.

Quality Assurance

Software Testing & Validation

Software bugs in a 3D printer can damage hardware, waste filament, and in the worst case, pose safety risks. Our software QA process is as rigorous as our hardware QC.

Automated Test Suite

CI/CD pipeline with automated firmware build, unit tests, and integration tests triggered on every commit. Hardware-in-the-loop (HIL) testing with physical printers in a test farm running scripted G-code sequences. Regression test suite covers: temperature control accuracy, motion system precision, power-loss recovery, filament sensor response time, and OTA update integrity.

Safety Validation

Thermal runaway protection tested with simulated thermistor failure (open circuit, short circuit, incorrect reading). Heater MOSFET failure mode testing. Watchdog timer verification. Emergency stop response time measurement. All safety-critical firmware paths reviewed via manual code audit in addition to automated testing. Test reports included in certification documentation packages.

Compatibility Testing

Firmware tested across all supported controller boards (STM32F103, STM32F407, ATmega2560, RP2040, ESP32). Touchscreen UI tested on 5+ display models (DWIN, Nextion, BTT TFT, Creality-style, MKS). Slicer profiles validated on Windows, macOS, and Linux. Mobile app tested on iOS 15+ and Android 10+ across 20+ device models via Firebase Test Lab.

Documentation

Every firmware release includes: changelog (features, fixes, known issues), API reference with endpoint documentation, configuration guide (printer.cfg or Configuration.h), troubleshooting guide (common errors and solutions), update instructions (SD card and OTA methods), and compliance test reports for certified variants. All documentation delivered in your brand template and in your required languages.

Long-Term Support

We provide ongoing firmware maintenance: bug fixes for the warranty period and beyond, slicer profile updates for new slicer versions, security patches for connected features, and feature backporting from newer product generations where applicable. LTS (Long Term Support) firmware branches available for enterprise and education partners who require stability over new features.
